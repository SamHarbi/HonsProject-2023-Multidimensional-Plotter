My name is Sameer and I am making a Web-Based Tool for visualising / creating charts of multi-dimensional data in a 3D scene. Visualising data is often an important first step in data analysis and can help indentify important patterns in data. Consider the following table below as an example:

That's a LOT of data isn't it? Finding a pattern manually would be a difficult and long process. Even more so for bigger datasets. but what if we tried charting the data in a 3D graph instead?

Considering that x, y, z correlate to the Protein, Sugar and Calories Amount- It is instantly obvious that there is a linear relationship in the data. Larger calories per serving correlate with larger sugar content, with protein amounts not affecting the outcome too much. Needless to say, this chart saved us a lot of time

My Project aim's to make it easy to explore and chart this kind of data

As part of a project like this it is important to have it iteratively tested by others to identify any shortcomings or confusing aspects from the user's side. This tool is designed to be usuable with any specialist knowledge beyond an understanding of how charts work mathematically.
